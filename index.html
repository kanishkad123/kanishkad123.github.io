Introduction

What I am all about.

I'm Kanishka, a Full-stack Engineer, experienced in building and maintaining softwares for E-Commerce, E-Learning and Enterprise Resource Planning domains.

Started my journey in Web application development using ASP.NET(MVC). Later, worked on .NET Core Web API microservice architecture with MSSQL server database and Angular 2 as the front-end technology. Eventually, I led my organisation to migrate to Angular CLI for an application built using Angular 5 dynamic forms, Multi-tenant .NET Web APIs, MSSQL Server for user management & MongoDB as the tenant database.

While working on Angular CLI, Node.js piqued my curiosity to use an alternative technology for server side programming, compelling me to build a small application on the MEAN stack.

Recently, I have developed a highly configurable B2B E-Commerce platform in ReactJS/Redux with server side rendering using NextJS, aiming towards Isomorphic JS & micro-frontend architecture. On the server side, the platform used Ocelot API gateway and microservices built in .Net as well as NodeJs with inter-microservice communication using RabbitMQ.

Along with drafting preliminary project scope and creating project plan with resource requirements, finalizing application work-flow and wireframing the User Interface are activities integral to my day to day work.
